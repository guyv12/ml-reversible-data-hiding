# Contributing Instructions

---

## Use Pull Requests (PRs)

Please submit your changes via PRs to maintain consistency.

---

## Follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0)

All commit messages should follow the Conventional Commits specification, for example:

```
fix(.gitignore): now includes dataset/samples

prev .gitignore ignored the samples directory as opposed to what was intended
```
